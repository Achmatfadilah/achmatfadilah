### 🎯 Snake Game 🐍

A classic snake game built with Python and Pygame!

**Features:**
- 🎮 Smooth controls with arrow keys
- ⭐ Score tracking and level system
- 🏆 High score display
- ⏸️ Pause functionality
- 🎨 Beautiful grid and colors

**Tech Stack:** Python, Pygame

**Controls:**
- Arrow Keys: Move snake
- P: Pause game

## 🎮 How to Run My Games

### Python Games (Snake, Pong, RPS)

```bash
# Install required library
pip install pygame

# Clone my game repository
git clone https://github.com/achmatfadilah/snake-game.git
cd snake-game

# Run the game
python snake_game.py
```

### HTML/JavaScript Games (Tic-Tac-Toe, Endless Runner)

```bash
# Clone the repository
git clone https://github.com/achmatfadilah/tic-tac-toe.git
cd tic-tac-toe

# Open in browser
# Just double-click index.html
# OR use Python HTTP server:
python -m http.server 8000
# Then open http://localhost:8000
```
